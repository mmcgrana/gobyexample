Go by Example

Go è un linguaggio di programmazione open source ideato per la creazione di software semplice, veloce e efficiente.

Go by Example è una introduzione pratica a Go che utilizza programmi di esempio commentati con adeguate spiegazioni. Dai un'occhiata al primo esempio o cerca nella lista intera qui di seguito.
