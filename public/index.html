Go by Example - ES

Go es un lenguaje de programación de código abierto diseñado para construir software simple, rápido y confiable. Por favor, lee la documentación oficial para aprender un poco sobre el código de Go, herramientas, paquetes y módulos.

Go by Example es una introducción práctica a Go utilizando programas de ejemplo anotados. Revisa el primer ejemplo o navega por la lista completa a continuación.

Esta es la traducción a español del proyecto Go by Example, creado por dominicode, comunidad enfocada a enseñar programación en castellano.
